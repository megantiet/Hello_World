### Who is BTS?
![BTS Map of the Soul 7 Jacket Shoot](https://ibighit.com/bts/images/bts/profile/profile-kv-m.png)

BTS, also know as Bangtan Sonyeondan, is a 7-member Korean boy band that debuted on June 13th, 2013.
